الأشهر 1-3

الأهداف الرئيسية:

  • • إعداد بيئة التطوير والخوادم المحلية
  • • تجميع مجموعة بيانات أولية للتدريب
  • • تدريب الفريق على الأدوات الأساسية

المشاريع المستهدفة:

  • أداة كشف البرمجيات الخبيثة الأساسية: نموذج تعلم آلي بسيط لتصنيف الملفات
  • محلل شهادات الأمان: أداة تحليل شهادات SSL/TLS للكشف عن الشهادات المزيفة

الأشهر 4-9

الأهداف الرئيسية:

  • • تطوير نماذج متقدمة وتدريبها
  • • بناء واجهات برمجية API للتكامل
  • • اختبار شامل وتحسين الأداء

المشاريع المتقدمة:

  • نظام كشف التسلل الشبكي (NIDS): نموذج يعتمد على تحليل تدفق البيانات الشبكية
  • محرك تحليل التهديدات الذكي: استخدام LLM لتحليل تقارير التهديدات واستخراج المؤشرات
  • أداة مسح الثغرات الآلية: دمج مع أدوات مثل Nuclei لتحسين الكشف
  • نظام مراقبة السلوكيات غير الطبيعية: تحليل سجلات النظام باستخدام خوارزميات التعلم غير المشرف
  • مولد قواعد SIEM الذكي: استخدام Llama لإنشاء قواعد كشف مخصصة
